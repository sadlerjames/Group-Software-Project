Info

Quizzes:

Quizzes can be created in the create a quiz panel. These quizzes can then be allocated as a daily quiz or used as part of a treasure hunt


Daily quiz:

A quiz can be allocated by using the daily quiz panel. This will allow you to allocate any quiz you have created to be the quiz of the day for all users for any day you choose


Treasure hunts:

To create a treasure hunt, you first need to create activities. These need a human-readable location name as well as the actual coordinates, and a task for the user to do at this location.

#

You can then link activities into a single treasure hunt, which will create a pdf you can print and stick up around campus for players to access
